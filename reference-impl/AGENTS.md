# Workspace Agent Instructions

This is an agentic workspace. You have access to the filesystem under `/workspace`.

## Available Tools
- **bun** — JavaScript/TypeScript runtime, package manager, test runner
- **git** — Version control
- **claude** — Claude Code CLI

## Guidelines
- Write files to `/workspace/`
- Use `bun` for running scripts and installing packages
- Commit meaningful changes with descriptive messages
- Keep the workspace organized
